Put the hero background video here.

Save your La Union clip as:  launion-hero.webm

The home page hero (index.html) plays it automatically (muted + looping).
Until a video is added, the hero just shows the red/blue gradient background,
so the page still looks fine without it.

Notes:
- The video file is NOT committed to the repo (see .gitignore). GitHub rejects
  files larger than 100 MB, and a hero clip should be small anyway.
- Keep it short (10-20 seconds) and compressed (ideally under ~10 MB) so the
  page loads fast. A 200 MB file is slow and is overkill for a background.
